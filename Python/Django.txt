教程：https://www.runoob.com/django/django-first-app.html

安装
cmd命令行：pip install django。安装后将“python目录\Lib\site-packages\django”，“python目录\Scripts”添加至环境变量

验证
python命令行：执行以下命令检查是否安装成功，可以输出正确的django版本号则安装成功
import django
django.get_version()

创建第一个项目
django-admin startproject HelloWorld。路径默认为C:\Users\Administrator目录
